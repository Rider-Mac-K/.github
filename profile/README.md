## Overview

Rider Advanced is a cross-platform .NET IDE from JetBrains that brings ReSharper's legendary code analysis and refactoring to a standalone development environment for C#, .NET, ASP.NET, Unity, and Unreal Engine development. It combines JetBrains' IDE platform with ReSharper's deep understanding of .NET code, providing the intelligence that .NET developers rely on without the Visual Studio dependency.

ReSharper-powered code analysis provides over 2,500 code inspections that detect errors, suggest improvements, and enforce coding standards across C#, VB.NET, XAML, and other .NET languages. The analysis understands .NET-specific patterns including async/await, LINQ, nullable reference types, and source generators. Quick-fix suggestions resolve detected issues with a single action.

Unity integration provides Unity-specific code analysis, performance suggestions, shader support, and integrated debugging that understands the Unity engine's lifecycle, serialization, and performance characteristics. The integration makes Rider the preferred IDE for Unity game developers who want better code intelligence than the Unity editor provides.

Debugging handles .NET applications, Unity games, ASP.NET web applications, and Unreal Engine projects with breakpoints, watches, and the interactive evaluation window. The debugger supports attach-to-process, remote debugging, and decompiled code debugging.

For .NET developers and game developers who want the code intelligence of ReSharper in a fast, cross-platform IDE, Rider Advanced is the .NET development environment that combines JetBrains' IDE expertise with the deepest .NET code understanding available.

## Performance

Rider balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
